# 📘 Estructuras de Datos - Unidad 1

## 📌 Descripción

Este repositorio contiene el desarrollo de la **Actividad de Aprendizaje Individual - Unidad 1** de la asignatura **Estructuras de Datos** de la **Universidad de Cartagena**.

En este proyecto se aplican los conceptos fundamentales de estructuras de datos utilizando los lenguajes **TypeScript** y **C#**, permitiendo comparar diferencias sintácticas y de paradigma entre ambos.

---

## 🎯 Objetivos

* Comprender el uso de **arreglos y matrices**.
* Implementar estructuras de datos usando **Struct / Record**.
* Aplicar **Programación Orientada a Objetos** mediante clases.
* Comparar diferencias entre **TypeScript y C#**.
* Analizar el comportamiento de los datos en memoria.

---

## 📚 Contenido del Proyecto

### 🔹 1. Struct / Record

* Definición de estructura con:

  * Nombre
  * Edad
  * Promedio
* Creación de 3 instancias
* Almacenamiento en arreglos
* Recorrido e impresión de datos
* Modificación de valores
* Comparación de implementación entre TypeScript y C#

---

### 🔹 2. Objetos (Clases)

* Clase `Estudiante` con:

  * Atributos: nombre, edad, promedio
  * Método: `mostrarInfo`
* Creación de instancias
* Uso de arreglos/listas de objetos
* Modificación mediante método `setPromedio`
* Ejecución de métodos sobre cada objeto

---

### 🔹 3. Comparativa

Análisis entre Struct/Record y Objetos en ambos lenguajes:

* Definición
* Mutabilidad
* Tipado
* Uso en memoria (stack vs heap)
* Diferencias entre paradigma tipado fuerte (C#) y flexible (TypeScript)

---

### 🔹 4. Ejercicio Integrador

* Implementación de:

  * Arreglos de objetos
  * Objetos que contienen matrices
* Desarrollo en **TypeScript y C#**
* Comparación de:

  * Sintaxis
  * Manejo de datos
  * Estructura del código

---

## 🛠️ Tecnologías Utilizadas

* **TypeScript** (Node.js / entorno de ejecución)
* **C#** (.NET)

---

## 🚀 Cómo ejecutar el proyecto

### ▶️ TypeScript

1. Instalar dependencias:

```bash
npm install
```

2. Ejecutar:

```bash
npx ts-node archivo.ts
```

---

### ▶️ C#

1. Compilar y ejecutar:

```bash
dotnet run
```
